Tecnologias do Sistema Gideon

Ícone Tecnologia Categoria Descrição Uso no Gideon
Amazon Web Services (AWS) Cloud Provider Plataforma de serviços de computação em nuvem abrangente e amplamente adotada. ClickBus utiliza AWS para sua infraestrutura analítica. Infraestrutura principal para todos os componentes do sistema.
Amazon S3 Armazenamento de Dados Serviço de armazenamento de objetos altamente escalável, durável e seguro. Usado pela ClickBus. Data Lake (Raw, Processed, Curated Zones), armazenamento de modelos e backups.
AWS DMS (Database Migration Service) Ingestão de Dados Serviço para migrar bancos de dados para a AWS de forma fácil e segura. Usado pela ClickBus. Ingestão de dados de bancos de dados relacionais dos clientes.
AWS Glue ETL/ELT & Governança Serviço de ETL totalmente gerenciado que facilita a preparação e carga de dados. Inclui o Glue Data Catalog. Usado pela ClickBus. Processamento ETL/ELT, catalogação de dados.
AWS Lambda Computação Serverless Serviço de computação serverless que executa código em resposta a eventos. Usado pela ClickBus. Processamento de dados orientado a eventos, micro-serviços.
Apache Spark Processamento de Big Data Framework de computação distribuída de código aberto para processamento de grandes volumes de dados. Transformações de dados complexas (via AWS EMR ou AWS Glue).
Python Linguagem de Programação Linguagem versátil e popular para desenvolvimento web, scripts, ciência de dados e machine learning. Desenvolvimento de scripts de ETL, modelos de ML, APIs.
Pandas (Python) Manipulação de Dados Biblioteca Python para manipulação e análise de dados de alto desempenho e fácil utilização. Limpeza, transformação e análise de dados em scripts Python.
Amazon SageMaker Machine Learning Plataforma completa para construir, treinar e implantar modelos de machine learning em escala. Ciclo de vida completo dos modelos preditivos (segmentação, previsão de compra, recomendação).
Scikit-learn (Python) Machine Learning Biblioteca Python com ferramentas simples e eficientes para mineração e análise de dados. Implementação de algoritmos de clusterização, classificação e regressão.
TensorFlow (Python) Machine Learning Plataforma de código aberto de ponta a ponta para machine learning. Desenvolvimento de modelos de deep learning (opcional, para problemas mais complexos).
PyTorch (Python) Machine Learning Biblioteca de machine learning de código aberto conhecida por sua flexibilidade e velocidade. Desenvolvimento de modelos de deep learning (opcional, alternativa ao TensorFlow).
XGBoost (Python) Machine Learning Biblioteca otimizada de gradient boosting, projetada para eficiência, flexibilidade e portabilidade. Modelos de classificação e regressão de alta performance.
Redis Banco de Dados em Memória Armazenamento de estrutura de dados em memória, usado como banco de dados, cache e message broker. Usado pela ClickBus. Cache de dados, armazenamento de predições para acesso rápido.
Amazon QuickSight Business Intelligence Serviço de BI escalável, serverless, incorporável e alimentado por ML. Criação e visualização de dashboards interativos.
AWS Athena Consulta de Dados Serviço de consulta interativa que facilita a análise de dados no Amazon S3 usando SQL padrão. Usado pela ClickBus. Consultas ad-hoc sobre o Data Lake.
Apache Airflow Orquestração de Pipeline Plataforma para criar, agendar e monitorar programaticamente fluxos de trabalho. Gerenciamento e automação dos pipelines de dados e ML.
AWS Step Functions Orquestração de Pipeline Serviço serverless para orquestrar fluxos de trabalho visuais para aplicações distribuídas. Alternativa ou complemento ao Airflow para orquestração na AWS.
Git Controle de Versão Sistema de controle de versão distribuído, gratuito e de código aberto. Gerenciamento de código-fonte do projeto.
Jira Gerenciamento de Projetos Ágil Ferramenta popular para rastreamento de bugs, issues e gerenciamento de projetos. ClickBus utiliza ferramentas Atlassian. Gerenciamento de tarefas, sprints e backlog do projeto.
Scrum Metodologia Ágil Framework para desenvolver e manter produtos complexos, com foco em colaboração e entrega iterativa. ClickBus adota práticas ágeis. Metodologia de desenvolvimento do projeto Gideon.
Kanban Metodologia Ágil Método para gerenciar o fluxo de trabalho, visualizando tarefas e limitando o trabalho em progresso. Gestão visual do fluxo de desenvolvimento e tarefas.
Slack Comunicação de Equipe Plataforma de mensagens e colaboração para equipes. Usado pela ClickBus. Comunicação interna da equipe de desenvolvimento e stakeholders.
